# Picsum Photos Viewer

Picsum Photos API를 사용하여 랜덤 이미지를 보여주는 간단한 웹 애플리케이션입니다.

## 기능

- 🖼️ 랜덤 이미지 표시
- 📐 이미지 크기 조절 (너비/높이)
- 🔄 새 이미지 로드
- 📸 특정 이미지 ID로 이미지 로드
- 💾 이미지 다운로드

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 너비와 높이를 조절하여 원하는 이미지 크기를 설정합니다.
3. "🔄 새 이미지" 버튼을 클릭하여 새로운 랜덤 이미지를 로드합니다.
4. "📸 특정 이미지" 버튼을 클릭하여 특정 ID의 이미지를 로드합니다.
5. "💾 다운로드" 버튼을 클릭하여 현재 표시된 이미지를 다운로드합니다.

## Picsum Photos API

- **랜덤 이미지**: `https://picsum.photos/{width}/{height}`
- **특정 이미지**: `https://picsum.photos/id/{imageId}/{width}/{height}`
- **예시**: `https://picsum.photos/300` (300x300 크기)

더 많은 정보는 [Picsum Photos 공식 사이트](https://picsum.photos/)를 참조하세요.
